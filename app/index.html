Hecho por Davel Gomoz

Una aplicación para escritores

LETRAS es una aplicación web que te ayudará a contar las palabras y carácteres de tu texto, así como a cambiar el formato del mismo.
Palabras:
0
Caracteres:
0
Elimina cualquier cantidad de saltos de línea de tu texto para reemplazarlos por solo uno y agrega una tabulación al inicio de cada párrafo. Al tocar el botón, el nuevo texto se copiará automáticamente en tu clipboard

¡Hola! Programé esta aplicación para que me ayudase a trabajar con los textos de mis novelas. Decidí publicarla en la web por si resultaba ser de ayuda para otros escritores, pero es posible que sus flujos de trabajo sean distintos al mío. ¿Tienes alguna idea para mejorar este pequeño proyecto?
